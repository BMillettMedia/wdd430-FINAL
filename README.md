# Persona Confidant Manager

A full-stack Angular + Node.js + MongoDB application that allows users to track Persona confidants across multiple games.

Supported Games
- Persona 3 Portable
- Persona 3 Reload
- Persona 3 Episode Aigis (DLC for Persona 3 Reload)
- Persona 4 Golden
- Persona 5 Royal

Features
- Full CRUD functionality
- MongoDB database
- Angular frontend
- Search and filtering
- Rank progress tracking
- Custom user notes
- Sorting options
- Persona themed UI

Technologies
- Angular
- Node.js
- Express
- MongoDB
- Mongoose

How to Run

Backend

cd server
npm install
node server.js

Frontend

cd client
npm install
ng serve

Then open

http://localhost:4200
